The Secretary shall require than any individual income tax return prepared by a tax return preparer be filed on magnetic media if (i) such return is filed by such tax return preparer, and (ii) such tax return preparer is a specified tax return preparer for the calendar year during which such return is filed. Subsection (e) of section 6011 of the Internal Revenue Code of 1986 is amended by adding at the end the following new paragraph: (3) Special rule for tax return preparers. (B) Specified tax return preparer. SECTION 1. SHORT TITLE. This Act may be cited as the Homebuyer Tax Credit Improvement Act of 2009. SEC.
